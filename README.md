**Key features:**

- User authentication
- Customize store logo, hero image, color theme
- Show order list, order details
- Add/Edit product detail



### Installation
`npm i` in these folders: 
- root folder(highstreet)
- /apps/admin
- /apps/storefront

### Running locally
- App : `npm run dev`
- Prisma Studio: `npx prisma studio` (in /packages/database)
